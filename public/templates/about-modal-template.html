What is the BGST?

BGST is an acronym that stands for "Board Game Session Tracker". In short, the BSGT is a cloud-based web application that allows user's to maintain their detailed board gaming history from any network accessible device.

What platforms does the BGST support?

At present, the BGST is supported on all current JavaScript enabled web browsers as well as iOS, Android, and BlackBerry devices.
